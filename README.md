# Suivi des Dépenses

Une application simple de suivi des dépenses pour gérer vos dépenses quotidiennes.

## Fonctionnalités

- Ajouter, modifier et supprimer des dépenses
- Catégoriser les dépenses
- Voir le résumé des dépenses
- Design réactif
